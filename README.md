A [BSzF_hf projekt](https://github.com/tarpeti111/BszfHF) (elméletileg) tartalmazza az összes forrás file-t amiből lehet build-elni az STK3700 EFM32 GiantGecko fejlesztői kártyára a játék programját.

A Simplicity Studio 5 Software Componet-ek amiket hozzá kell adni a projekthez:

	- Peripherals/LCD,
	- Peripherals/ADC,
	- Peripherals/USART
